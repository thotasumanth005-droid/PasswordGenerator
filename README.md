🔐 Password Generator & Signup App

A simple web app to generate strong passwords and create an account.

🚀 Features
🔑 Generate random password
⚙️ Choose options (uppercase, lowercase, numbers, symbols)
📏 Adjust password length
📊 Check password strength
📋 Copy password
📧 Signup with email and password
🛠️ Technologies Used
HTML
CSS
JavaScript
📂 Files
signup.html – Main page
script.js – Logic & functions
style.css – Styling
